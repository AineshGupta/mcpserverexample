# Installation Steps

To install the add_tool MCP Server using Git deployment, run the following command:

```json
{
  "mcpServers": {
    "add_tool": {
      "command": "uvx",
      "args": [
        "--from",
        "git+https://github.com/AineshGupta/mcpserverexample.git",
        "mcp-server"
      ]
    }
  }
}
```
